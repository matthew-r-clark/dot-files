# Session Naming

When a Jira ticket is clearly the primary focus of the current session (whether referenced directly or derived from a Bitbucket PR, branch name, or similar), update the session name to:

```
[<ticket-number>] <short description>
```

Keep the description concise. Only do this when confident the ticket is the central subject of the work — not for incidental mentions.

Do not rename a session if it already has a name.
